## 新建脚本文件夹
- 选择一个位置创建文件夹

- 修改编辑器设置`settings.json`

```json
"code-recycle.script": {
    "dir": "/path/to/code-recycle-plugin-script"
}
```


?> 可以参考[演示仓库](https://github.com/wszgrcy/code-recycle-plugin-script)

- 按照以下层级创建

```tree
.
├── action // 执行动作
│   ├── hello2.js
│   └── hello.js
├── snippet // 代码片段
│   ├── hello2.js
│   └── manifest.json
└── view //视图
    ├── hello2.js
    └── hello.js
```
